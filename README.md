# AROS Getting Started

Beginner-friendly, fully verified guides for AROS x86_64 (ABIv11)
development: building the toolchain, running AROS One in VirtualBox,
cross-compiling C programs, and learning Zune.

Published at: https://elwis.github.io/aros-getting-started/

Every command in these guides has been run and every path verified
against real build output before being written down.
